Proyectos destacados

EduTech — Plataforma e-learning full-stack (TFG)

Monorepo con npm workspaces que integra un frontend en React + Vite y un backend en Express con base de datos PostgreSQL. Incluye autenticación con Google OAuth 2.0 (Passport) y cuentas locales con contraseña hasheada, control de acceso por roles (alumno/profesor/administrador), CRUD completo de cursos, lecciones y tests, y un backoffice de administración. La base de datos corre en Neon (PostgreSQL serverless) y el backend está desplegado en Render.

⚙️Tecnologías: React • Vite • Express • PostgreSQL • Passport (OAuth 2.0) • Neon • Render

Event App - Sistema Gestor de Eventos en Tiempo real

Aplicación web diseñada para la gestión y visualización de eventos en tiempo real, entre cliente y servidor. La aplicación permite filtrar por categoría y fechas, e inscribirse en los eventos con actualización inmediata del contador de participantes. Se implementó una arquitectura modular separando backend (API REST + WebSocket) de frontend (React con Context API), asegurando escalabilidad y mantenibilidad. La interfaz incluye efectos visuales como transiciones suaves, feedback interactivo en botones, y diseño adaptable a dispositivos móviles, y desktop.

⚙️Tecnologías: Node.js • Express • MongoDB • Socket.IO • React • Tailwind CSS • Context API • AJAX • WebSockets

Guía Interactiva de Puntos de Interés — España

Aplicación web 100% cliente para explorar puntos de interés geolocalizados en España. Cada punto incluye una guía multimedia con reproductor de audio/vídeo HTML5, y un mini-mapa real con Leaflet y OpenStreetMap. Calcula la distancia a cada lugar mediante la fórmula de Haversine a partir de la ubicación del usuario (con «Detectar mi ubicación», procesada siempre en local, nunca enviada a un servidor). Incluye un formulario para añadir nuevos puntos de interés (nombre, categoría, descripción, coordenadas y multimedia), diseño responsivo con CSS Grid y `clamp()`, y roles ARIA para accesibilidad.

⚙️Tecnologías: HTML5 • CSS3 • JavaScript (vanilla) • Leaflet • OpenStreetMap • Geolocation API
